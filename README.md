# cinecalidad-scraper

This is a simple script for scraping movies data from cinecalidad.
You decide how you want to save the data.

## Required libraries
- selenium
- fake_useragent
- requests
- pillow

## If want to automitize you can install
- schedule
## Required Webdriver
- Chrome
https://chromedriver.chromium.org/
